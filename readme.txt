Per visualizzare in modo agevole il codice del progetto e i relativi output, è stato realizzato il notebook "movierecsys.ipynb" utilizzabile su Google Colab.

Per eseguire l'interfaccia grafica, è necessario utilizzare i file python presenti nella cartella "File", assicurarsi di aver installato sul proprio dispositivo la libreria python "Streamlit", ed eseguire il seguente comando all'interno della cartella:
streamlit run movierecsys_app.py
